# I2COMSAPP 2026 - Paper Templates & Downloads

## Instructions for Adding Template Files

Place the following template files in this directory (`public/downloads/`):

### Required Files:

1. **I2COMSAPP_Word_Template.zip**
   - Contains: Microsoft Word template (.docx) with Springer LNNS formatting
   - Use this if you prefer to write in Microsoft Word

2. **I2COMSAPP_LaTeX_Template.zip**
   - Contains: LaTeX source files (.tex) with all required packages and formatting
   - Use this if you prefer to write in LaTeX

3. **I2COMSAPP_Sample_Paper.pdf**
   - PDF example showing the correct format and layout
   - Reference this to understand the expected appearance

4. **I2COMSAPP_Complete_Package.zip**
   - All-in-one bundle containing:
     - Word template
     - LaTeX template
     - Submission guidelines (PDF)
     - Sample formatted papers
     - Author instructions

## File Specifications:

### For Word Templates (.zip):
```
I2COMSAPP_Word_Template.zip
├── I2COMSAPP_2026_Template.docx
├── Submission_Guidelines.pdf
└── Author_Instructions.txt
```

### For LaTeX Templates (.zip):
```
I2COMSAPP_LaTeX_Template.zip
├── main.tex
├── preamble.tex
├── references.bib
├── Springer_LNNS_Guide.pdf
└── Sample_Figures/
```

### For Sample Paper (.pdf):
- Well-formatted example paper showing proper:
  - Title formatting
  - Abstract structure
  - Section organization
  - Figure placement
  - Reference formatting
  - Page layout

## Access the Downloads:

The templates are available on the Call for Papers page:
- Direct link: `yoursite.com/downloads/`
- Call for Papers page: Shows all available templates with descriptions

## Important Notes:

1. **PDF Submission**: Authors must submit papers in PDF format only
2. **Formatting**: All papers must follow Springer LNNS guidelines
3. **Page Limit**: 8-15 pages including references
4. **Language**: English (French or Arabic abstracts welcome)
5. **Format**: A4, 2.5 cm margins, single column

## Steps to Use:

1. Download the appropriate template (Word or LaTeX)
2. Extract the .zip file
3. Open the template in your preferred editor
4. Follow the author instructions included
5. Write your paper using the template
6. Export/Compile to PDF
7. Submit the PDF through EasyChair or email
